# Aplicación Web IMC - Java EE

Este repositorio contiene el código fuente de la aplicación web para el cálculo y monitoreo del Índice de Masa Corporal (IMC), desarrollada como Evidencia Final para el curso de Computación Avanzada en Java (Universidad Tecmilenio).

## Requerimientos Cumplidos (Avance 1 y Final)
- ✅ Estructura de proyecto Maven
- ✅ Arquitectura MVC
- ✅ Persistencia de datos con MySQL
- ✅ Servicios REST (JSON y XML)
- ✅ Gestión de ramas en Git (master, develop, features)

## Tecnologías Utilizadas
- Java EE 8
- Servlets y JSP
- JAX-RS (Jersey) para REST
- MySQL
- HTML/CSS/Bootstrap para la interfaz
- Servidor: GlassFish

## Instrucciones de Despliegue
1. Clonar el repositorio.
2. Ejecutar el script SQL incluido en la carpeta `database` para crear la base de datos.
3. Abrir el proyecto en NetBeans.
4. Construir el proyecto con Maven (`Clean and Build`).
5. Desplegar en el servidor GlassFish.
